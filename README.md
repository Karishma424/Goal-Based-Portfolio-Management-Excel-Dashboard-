# Goal-Based-Portfolio-Management-Excel-Dashboard-
## Project Overview

This project is an interactive Goal-Based Portfolio Management Dashboard built entirely in Microsoft Excel. It helps users plan, track, and simulate investments for multiple financial goals while monitoring portfolio performance through dynamic dashboards, KPI cards, and visualisations.

The dashboard combines financial planning concepts with Excel analytics to provide a comprehensive investment management solution.

---

## Objective

- Plan investments based on multiple financial goals
- Allocate assets according to investor risk profile
- Track portfolio growth using SIP and lump sum investments
- Simulate future portfolio performance
- Monitor goal achievement through interactive dashboards
- Support informed financial decision-making

---

## Key Insights

- Visual tracking of financial goal progress
- Risk-based portfolio diversification
- Future value estimation using inflation
- Portfolio growth through SIP investments
- Funding ratio analysis for each goal

## How to Use

1. Download the Excel workbook.
2. Open using Microsoft Excel (2019 or Microsoft 365 recommended).
3. Navigate through the different worksheets.
4. Enter or modify investment data.
5. Explore the interactive dashboard, charts, and KPI cards.
6. Analyse portfolio performance and financial goal progress.
